# Angular Movie App Database

##Features
  * User Login/Logut (with Bootstrap theme)
  * User Registration
  * User Movie Search (through OMDB database)
  * Add Movie to your personal collection. 
  * Search your personal collection using Actor, Genre, or Title.


### This is a simple angular SPA which uses angularjs on the front and and uses Express JS on the backend. 

### Steps.
#### Change the database settings at config/database.js
#### npm install
#### npm start
#### visit http://localhost:3000
